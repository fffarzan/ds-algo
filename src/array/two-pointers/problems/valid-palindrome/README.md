# Problem

Given a string `str`, return `true` if it is a palindrome, otherwise return `false`.

A palindrome is a string that reads the same forward and backward. It is also case-insensitive and ignores all non-alphanumeric characters. Alphanumeric characters consist of letters (A-Z, a-z) and numbers (0-9)

Hint: We should get rid of alphanumerical characters like space.

# Metrics

- Time Complexity: O(n)
- Space Complexity: O(1)
