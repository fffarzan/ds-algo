# Problem

Given an integer array `nums`, return all the triplets `[nums[i], nums[j], nums[k]]` where `nums[i] + nums[j] + nums[k] == 0`. Indices `i`, `j` and `k` are distinct.

# Metrics

- Time Complexity: O(nlogn) + O(n^2) => o(n^2)
- Space Complexity: O(1) / O(n) <== depends on sorting algorithm we used.
